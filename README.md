# library-management-System
Library Management System: (Java + JDBC)
This project focuses on implementing CRUD operations and managing relational data using Java and MySQL. This system allows library administrators to manage a collection of books and a database of members. It handles the core logic of issuing books, tracking inventory, and maintaining member records through a seamless connection between Java and a relational database.

Tech Stack
Language: Java (JDK 17+)
Database: MySQL 8.0
Connectivity: JDBC (Java Database Connectivity)
IDE: IntelliJ IDEA
Build Tool: Maven

Features
-Book Management: Add new titles, view the full catalogue, and remove outdated records.
-Member Directory: Register new library members with unique auto-generated IDs.
-Issue System: (In Progress) Link members to books with automated inventory updates and date tracking.
-Data Persistence: All records are stored securely in a MySQL schema.

Things I got to learn:
- How classes interact with each other, OOPS concepts.
- Java Database Connectivity: Connected my application to an existing database.
- Usage of PreparedStatements to prevent SQL Injection.
